## Description

New Zealand government strategy for procuring secure and resilient cloud services across agencies. Establishes a consolidated Marketplace channel for IaaS and cloud services, introduces security tiering, supports multi-cloud and hybrid approaches, and incorporates Māori data sovereignty and Te Tiriti considerations into cloud procurement.
