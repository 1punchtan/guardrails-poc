## Description

New Zealand government standard requiring public service agencies to implement controls and legal agreements when sharing personal information with non-government third parties delivering public services. Mandatory from 1 July 2025, it establishes best practice approaches for information sharing agreements that protect personal data and ensure adequate safeguards.
