## Description

New Zealand legislation establishing the governance framework for the State Sector, defining roles and responsibilities of the State Services Commissioner, chief executives, and public service personnel. This Act (now repealed as of August 2020 by the Public Service Act 2020) established foundational principles for public service management, employment conditions, and organizational structure.
