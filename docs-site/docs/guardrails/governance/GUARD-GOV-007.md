## Description

A comprehensive policy framework for the respectful, trusted, and transparent use of personal information in the social sector. Establishes five principles (He Tāngata, Manaakitanga, Mana Whakahaere, Kaitiakitanga, Mahitahitanga) and four guidelines (Purpose Matters, Transparency and Choice, Access to Information, Sharing Value) to guide agencies in data collection, use, and sharing practices.
