## Description

New Zealand legislation establishing the statutory framework for collecting, managing, and accessing official statistics and data. Defines roles of Statistics New Zealand and the Government Statistician, governs data collection practices, statistical confidentiality, research access, and enforcement mechanisms for public sector data governance.
