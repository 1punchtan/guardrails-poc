## Description

A comprehensive framework designed by Māori data experts for the Aotearoa New Zealand public service to govern Māori data as a taonga. The model establishes eight pillars covering data capacities, infrastructure, collection, protection, access, use, quality, and classification, grounded in Te Tiriti o Waitangi and focused on enabling Māori self-determination and reducing unethical data use.
