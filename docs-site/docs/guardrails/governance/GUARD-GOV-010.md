## Description

New Zealand Government policy framework for open access and licensing of government-owned software under free and open source software (FOSS) licenses. Provides legal context, policy principles, guidance on selecting appropriate FOSS licenses (permissive vs. copyleft), and a structured review and release process to ensure legally robust public software releases.
