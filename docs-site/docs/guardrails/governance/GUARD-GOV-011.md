## Description

New Zealand code of practice governing how telecommunications agencies must collect, store, use, and disclose telecommunications information and subscriber data. Establishes 13 privacy rules covering lawful purpose, data security, individual access rights, accuracy, retention, and cross-border disclosure of telecommunications and traffic information.
