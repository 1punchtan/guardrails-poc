## Description

Mandatory standard for all NZ public service departments and Executive branch organizations requiring WCAG 2.2 Level AA compliance for publicly and internally facing web pages, effective 17 March 2025. Includes specific exemptions and modifications for complex visual content, live audio/video, and inactive pages.
